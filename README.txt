Magic Card Creator
==================

A standalone desktop app for designing custom Magic: The Gathering style cards.

HOW TO OPEN
-----------
1. Double-click "index.html" to open it in your web browser
   (Chrome, Firefox, Safari, or Edge all work great)

No installation needed — it runs entirely in your browser.

FEATURES
--------
- Live card preview as you type
- 7 color themes: White, Blue, Black, Red, Green, Colorless, Multicolor
- Mana cost with colored symbols: {W} {U} {B} {R} {G} {C} {T} {1}–{20}
- Card text formatting:
    *text*         → italic
    ~              → replaced with the card's name
    *"text"*       → flavor text with divider line
    {W}{U}{B}...   → inline mana symbols
- Keywords auto-bolded (Flying, Trample, Hexproof, etc.)
- Creature Power / Toughness box
- Full Art mode: image fills entire card with frosted-glass text overlay
- Upload your own card art image
- Auto-scaling text: font shrinks to fit the text box
- Download card as high-resolution PNG (3× scale)

NOTE
----
Requires an internet connection the first time to load Google Fonts
(Cinzel + Crimson Text). After that, fonts may be cached by your browser.
